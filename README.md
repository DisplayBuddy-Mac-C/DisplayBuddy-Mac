## 💡 Overview

<a href="#displayBuddy">DisplayBuddy</a> serves Mac users with external monitors who are frustrated by the inaccessibility of monitor controls — having to use the monitor's physical OSD buttons every time brightness adjustment is needed, losing track of the menu when looking for input source, or lacking keyboard shortcut control for external display brightness.

The <a href="#displayBuddy">developer and creative professional use case</a> involves frequent brightness adjustment throughout the day — lowering brightness in the evening, raising it for detailed color work, adjusting for ambient lighting changes — all workflows that become tedious when requiring physical button presses on the monitor. DisplayBuddy makes these adjustments as quick as adjusting laptop brightness. The <a href="#displayBuddy">multi-monitor setup use case</a> controls brightness across multiple external monitors — setting different brightness levels for monitors in different positions, dimming a secondary reference monitor while keeping the primary at working brightness, and managing the display environment for extended sessions.

The <a href="#displayBuddy">multi-device monitor use case</a> serves users whose monitors connect to multiple devices and require frequent input source switching — changing from Mac to another device and back through a keyboard shortcut or menu click rather than navigating the physical OSD each time. <a href="#displayBuddy">Eye strain and comfort use case</a> makes brightness adjustment responsive enough to actually use throughout the day — fine-tuning external display brightness to match ambient light changes, time of day, and content type without the friction that physical controls impose.</p>

<a href="#displayBuddy">Night mode scheduling</a> in some configurations automates brightness reduction at configured times — the display automatically dims in the evening without requiring manual adjustment. <a href="#displayBuddy">Compatibility note</a> — DDC control requires monitors and connections that support the DDC protocol. Most modern monitors on HDMI, DisplayPort, and USB-C connections support DDC, but specific compatibility depends on the monitor model and cable type.

<a href="#displayBuddy">Lightweight menu bar utility design</a> keeps DisplayBuddy small and non-intrusive — a focused tool that does monitor control well without feature clutter. <a href="#displayBuddy">Regular updates for macOS compatibility</a> maintain DisplayBuddy's DDC communication reliability as macOS updates change system-level display handling.

## 📌 FAQ

<details>
<summary>Why doesn't DDC work with my monitor?</summary>

DDC compatibility depends on the specific monitor model, the cable type, and the connection path. Some monitors don't support all DDC commands, some USB-C and USB hubs block DDC, and some cable types don't carry DDC signaling.

</details>

<details>
<summary>Does DisplayBuddy have a menu bar display?</summary>

Yes. A compact menu bar panel shows controls and current settings for connected monitors.

</details>

<details>
<summary>What display information can DisplayBuddy show?</summary>

Current brightness level, input source, volume, and monitor model information read through DDC.

</details>

<details>
<summary>Is DisplayBuddy a one-time purchase or subscription?</summary>

DisplayBuddy is available through Setapp subscription. Check the DisplayBuddy website for direct purchase options.

</details>

<details>
<summary>Does DisplayBuddy work with USB-C monitors?</summary>

Yes, when the USB-C connection and monitor support DDC. Compatibility varies by monitor model and connection type.

</details>
